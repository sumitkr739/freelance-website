1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

By default the server starts at `http://127.0.0.1:3000`. If that port is already busy, it automatically tries the next available port. You can also set `PORT` or `HOST` manually.
